# LemGen AI Design MCP

Ein MCP-Server für visuelle KI-Erstellung in Claude Code, Cursor, Codex, Windsurf, Roo Code, OpenClaw und jedem MCP-kompatiblen Host.

- 4.533 kuratierte Bild- und Videoprompts
- Prompt-Übersetzung und Verbesserung in 18 Sprachen
- Bibliothek für GPT Image, Nano Banana, Seedance und Midjourney
- Bild-/Videogenerierung über die LemGen API

## Quick Start

```json
{
  "mcpServers": {
    "lemgen": {
      "command": "npx",
      "args": ["-y", "github:aithink001/lemgen-ai-design-mcp"]
    }
  }
}
```

Für Bild- oder Videogenerierung `LEMGEN_API_TOKEN` setzen.
