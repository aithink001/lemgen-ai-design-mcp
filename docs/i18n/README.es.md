# LemGen AI Design MCP

Un servidor MCP para creación visual con IA en Claude Code, Cursor, Codex, Windsurf, Roo Code, OpenClaw y cualquier host compatible con MCP.

- 4,533 prompts curados de imagen y video
- Traducción y mejora de prompts en 18 idiomas
- Biblioteca para GPT Image, Nano Banana, Seedance y Midjourney
- Generación de imagen/video con la API de LemGen

## Quick Start

```json
{
  "mcpServers": {
    "lemgen": {
      "command": "npx",
      "args": ["-y", "github:aithink001/lemgen-ai-design-mcp"]
    }
  }
}
```

Para generar imagen o video, configura `LEMGEN_API_TOKEN`.
