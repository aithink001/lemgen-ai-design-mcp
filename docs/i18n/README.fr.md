# LemGen AI Design MCP

Un serveur MCP pour la création visuelle IA dans Claude Code, Cursor, Codex, Windsurf, Roo Code, OpenClaw et tout hôte compatible MCP.

- 4 533 prompts image/vidéo sélectionnés
- Traduction et amélioration de prompts en 18 langues
- Bibliothèque pour GPT Image, Nano Banana, Seedance et Midjourney
- Génération image/vidéo via l'API LemGen

## Quick Start

```json
{
  "mcpServers": {
    "lemgen": {
      "command": "npx",
      "args": ["-y", "github:aithink001/lemgen-ai-design-mcp"]
    }
  }
}
```

Pour générer des images ou vidéos, configurez `LEMGEN_API_TOKEN`.
