# LemGen AI Design MCP

Claude Code、Cursor、Codex、Windsurf、Roo Code、OpenClaw などの MCP 対応エージェント向けの AI 画像・動画制作ツールです。

- 4,533 件のキュレーション済み画像/動画プロンプト
- 18 言語のプロンプト翻訳・改善
- GPT Image、Nano Banana、Seedance、Midjourney のプロンプトライブラリ
- LemGen API による画像/動画生成

## Quick Start

```json
{
  "mcpServers": {
    "lemgen": {
      "command": "npx",
      "args": ["-y", "github:aithink001/lemgen-ai-design-mcp"]
    }
  }
}
```

画像/動画生成には `LEMGEN_API_TOKEN` が必要です。
