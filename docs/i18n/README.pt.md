# LemGen AI Design MCP

Um servidor MCP para criação visual com IA no Claude Code, Cursor, Codex, Windsurf, Roo Code, OpenClaw e qualquer host compatível com MCP.

- 4.533 prompts curados de imagem e vídeo
- Tradução e melhoria de prompts em 18 idiomas
- Biblioteca para GPT Image, Nano Banana, Seedance e Midjourney
- Geração de imagem/vídeo pela API da LemGen

## Quick Start

```json
{
  "mcpServers": {
    "lemgen": {
      "command": "npx",
      "args": ["-y", "github:aithink001/lemgen-ai-design-mcp"]
    }
  }
}
```

Para gerar imagem ou vídeo, configure `LEMGEN_API_TOKEN`.
